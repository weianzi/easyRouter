# easyRouter

利用微信团队开发的[Router](https://github.com/progrape/router)，做的一个简单的、无刷新读取新闻列表和新闻详情。仅支持高级浏览器和手机浏览器。


### 预览
![image](https://github.com/weianzi/easyRouter/blob/master/news.gif)



###项目运行
```
npm i 安装依赖
npm start 开始
npm run build 合并打包
```
